## Workflow

1. **Write a 1-line design read**
   - “Reading this as: <page kind> for <audience>, with a <vibe> language, leaning toward <aesthetic family>.”
2. **Set the 3 dials**
   - `DESIGN_VARIANCE` (symmetry → artsy)
   - `MOTION_INTENSITY` (static → cinematic)
   - `VISUAL_DENSITY` (airy → packed)
3. **Kill the defaults**
   - No generic “three equal feature cards”, no random purple glow gradients, no glass everywhere.
4. **Pick 1 coherent system**
   - One type scale, one spacing rhythm, one radius system, one accent strategy.
5. **Ship with a pre-flight pass**
   - Check readability, CTAs, mobile layout, motion fallbacks, and “could this be an AI template”.

## Anti-slop guardrails

- Make the hero fit the viewport (CTA visible, no bloated copy).
- Avoid repeated section scaffolds (eyebrow labels everywhere, same grid everywhere).
- Prefer real visual moments (3D scene, real imagery) over “CSS blob filler”.
- Reduced motion is mandatory when motion intensity is high.
