## Practical workflow

1. Clarify the product/page type (portfolio, landing, dashboard, etc.) and audience.
2. Decide on style family + constraints (a11y, perf, motion limits).
3. Apply priority order:
   - Accessibility, interaction/touch, performance
   - Layout/responsive, typography/color
   - Motion and micro-interactions
4. Implement changes consistently (tokens/radii/shadows/motion rhythm).
5. Verify on mobile + reduced motion.
